How to Add a Publication

01

Create a folder

In publications/, create a folder named exactly after your PDF (without the .pdf extension). Example: publications/My-Research-2024/

02

Add your files

Place two files inside that folder:
• My-Research-2024.pdf — the publication itself
• cover.jpg — the cover image (portrait ratio recommended)

03

Register the publication

Open publications.json at the root and add a new entry. The title will be taken from the filename automatically.

04

Push to GitHub

Commit and push. GitHub Pages will rebuild automatically — your new publication will appear on the site within minutes.
